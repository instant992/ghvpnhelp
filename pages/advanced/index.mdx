## Маршрутизация в NekoRay

Для для того, что бы настроить правила работы сайтов или приложений с ghostVPN в системе Windows, воспользуйтесь клиентом NekoRay и этой статьей с набором инструкций.

У приложения есть два режима работы. По белому и черному списку. Белый список, это список сайтов и приложений, которые используют VPN-подключение, а весь остальной трафик идет мимо. Черный список, соответственно, настраивает сайты и приложения, которые не будут использовать VPN-подключение.

## Таблица маршрутизации

Маршрутизация NekoRay проводится в разделе <b>«Настройка» → «Настройка маршрутов» → «Базовые маршруты»</b>.

В <b>таблице</b> указываются правила для доменов и IP-адресов.
![Alt text for the image](https://vpnghost.space/manual/neko1.avif)

<b>Столбцы</b>

Напрямую – сайты идут мимо VPN, Прокси – все сайты открываются с использованием VPN, Блок – сайты не открываются.

<b>Строка Домен</b>

В строке <b>Домен</b> вносятся сайты в формате domain:site.com. Например, базовый список сайтов, которые могут пригодиться:
>- domain:youtu.be 
>- domain:rutracker.org 
>- domain:youtube.com
>- domain:streamable.com
>- domain:ytimg.com
>- domain:googlevideo.com
>- domain:googleapis.com
>- domain:ggpht.com
>- domain:chatgpt.com
>- domain:chat.openai.com
>- domain:instagram.com
>- domain:twitter.com
>- domain:facebook.com
>- domain:x.com
>- domain:linkedin.com
>- domain:twimg.com
>- domain:fbcdn.net

<b>Строка IP</b>

В строке IP вносятся адреса и сегменты сети в формате geoip. Например, правила для того, чтобы маршрутизировать все российские сайты:
>- geoip:ru
>- geoip:private

<b>Что такое режим Outbound?</b>

Это политика доступа для всех сайтов, которые не указаны в таблице правил.
Proxy – все сайты идут с использованием VPN, Bypass – сайты идут мимо VPN, Block – сайты не открываются.


<b>Кастомные маршруты</b>

Во вкладке Кастомные маршруты прописываются правила маршрутизации сайтов и приложений в формате JSON.

## Настройка VPN для сайтов и приложений

Настройка сайтов и приложений в режиме proxy, все остальное подключается напрямую мимо VPN

1. В приложении NekoRay нажмите «Настройка» → «Настройка маршрутов» → «Базовые маршруты» → «Кастомные маршруты».

2. Вставьте JSON конфигурацию. Например, для эти правила для Discord и популярных браузеров:

```jsx {} showLineNumbers
{
    "rules": [
        {
            "outbound": "proxy",
            "process_name": [
                "Discord.exe",
                "chrome.exe",
                "firefox.exe",
                "msedge.exe",
                "opera.exe",
                "browser.exe",
                "yandexbrowser.exe",
                "brave.exe",
                "vivaldi.exe",
                "librewolf.exe"
            ]
        }
    ]
}
```

3. «Outbound по умолчанию» выберите режим bypass

4. Примете изменения нажав ОК

5. Активируйте «Режим TUN» и подключитесь к нужной локации.

При необходимости внести правила маршрутизации сайтов или сегментов сети в таблицу.

## FAQ

<b>Как найти название процесса для правил</b>

Перейдите в свойства приложения. Во вкладке «Общее» или «Ярлык» будет написано название приложение и тип файла. Также в свойствах ярлыка указано название объекта.
![Перейдите в свойства приложения. Во вкладке «Общее» или «Ярлык» будет написано название приложение и тип файла. Также в свойствах ярлыка указано название объекта.](https://vpnghost.space/manual/neko1.avif)

<b>Как исключить российский трафик из работы VPN</b>
Чтобы российские сайты открывались без VPN

1. В приложении NekoRay нажмите «Настройка» → «Настройка маршрутов» → «Базовые маршруты» → «Кастомные маршруты».

2. Вставьте JSON конфигурацию. Например, для эти правила для маршрутизации всего российского трафика напрямую мимо VPN

```jsx {} showLineNumbers
{"rules":[{"domain_suffix":[".ru"],"outbound":"direct"},{"outbound":"proxy"}]}
```

3. В таблице маршрутизации пропишите эти правила

IP напрямую:
>- geoip:ru 
>- geoip:private

Домен напрямую:
>- geosite:ru

4. «Outbound по умолчанию» выберите режим proxy

5. Примете изменения нажав ОК

6. Активируйте «Режим TUN» и подключитесь к нужной локации.
![Активируйте «Режим TUN» и подключитесь к нужной локации.](https://vpnghost.space/manual/neko4.avif)
